Why am I seeing this?

You currently have Focuser running in the background, and the timer is activated. This means all websites you have specified in the URLs page will display the same message.

So how can I access {{%webpage%}}?

You are probably seeing this page for a good reason (after all, Focuser is meant to help you.. focus!).
If you still wish to continue to the original content, you can either pause the timer or close the app from the system tray.
